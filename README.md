# fractol - Simple fractal rendering
![Screenshot](https://i.imgur.com/A9mhp9j.png)

Written in C with minilibX. Runs on linux and freeBSD.

## Dependencies
```
sudo apt update && sudo apt install gcc make xorg libxext-dev libbsd-dev
```

## Fractals
Mandelbrot
```
./fractol m
```
Julia
```
./fractol j <r> <i>
```
